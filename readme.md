# maimai tools

> 为国服音游玩家提供一个在线工具集锦。

## 路由说明

路由  |  功能    |  描述
------|---------|--
/hall | 机厅管理页面 | 已完成雏形，待优化
/about | 关于页面 | 对本站的描述
/song | 查歌页面 | 待完善
/profile | 用户绑定页面 | 待完善(优先)：通过绑定用户的QQ号来对评论、记录修改、权限、查分做基础
/dev | 开发者页面 | 一些开发工具
POST@/api | api相关功能 | 向其发送特定格式的JSON请求以对服务器进行通讯




------------------------------------

### 代码风格:

#### Javascript
- 属性/一般对象名/变量名, 使用下划线分割命名;
- 类, 使用大驼峰命名;
- 类的实例和函数, 使用小驼峰命名法;
- 对于代码书写, 不使用末尾分号;
- 对于每个 函数/类 需进行JSDoc注释以便开发;
